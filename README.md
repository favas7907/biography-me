# Favas M - Personal Biography Page

A premium, editorial-inspired personal biography page built using modern **HTML and CSS**. 


## 🧑‍💻 About

This repository contains the personal biography page of **Favas M.**, a Computer Science Engineering student with a strong interest in scalable technology solutions, focusing on Data Engineering, databases, and backend systems. 

The UI features a clean, high-end "Editorial Aesthetic" with a crisp white theme, premium blue accents, and dark typography for strong readability. It demonstrates advanced CSS layout techniques, semantic HTML structuring, and polished UI/UX design principles.

## 🚀 Technologies Used

- **Markup**: HTML5 (Semantic Structure)
- **Styling**: Modern CSS (Custom Properties, Flexbox, Grid, Hover Effects)
- **Design Strategy**: Utility-driven styling patterns
- **Typography:** Web Fonts (Playfair Display, Inter)
- **Icons**: SVG-based UI Icons

## 🎨 Design Philosophy

The design takes inspiration from premium design systems and editorial layouts, emphasizing:
- **Spaciousness**: Generous padding and margins for a breathable layout.
- **Typography**: High-contrast pairings. Display fonts for bold headers and clean sans-serif for highly readable body text.
- **Visual Hierarchy**: Clear separation of sections using soft borders, elegant card layouts, and subtle hover interactions.
- **Responsiveness**: Fluid adaptation from mobile screens to wide desktop monitors using CSS grid configurations and responsive spacing.
